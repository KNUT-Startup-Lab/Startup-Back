# CampusMate (캠퍼스메이트)

## 📝 프로젝트 소개
복잡한 기숙사 생활, 터치 한 번으로 해결하세요.
모바일 체크인부터 공용 냉장고 관리까지, 기숙사 거주 학생들의 실제 불편함을 해소하기 위해 개발된 생활 밀착형 솔루션입니다.

- **프로젝트 명:** CampusMate (캠퍼스메이트)
- **개발 기간:** 2025.09 ~ 2025.12
- **주요 타겟:** 기숙사 거주 학생 및 사생 관리실

## 💡 개발 배경 (Why?)
1. **아날로그식 관리 체계:** 체크인 및 상벌점 관리가 수기로 이루어지는 등 행정적 비효율성이 존재했습니다.
2. **공용 공간의 갈등:** 공용 냉장고 내 물품 분실 및 유통기한 관리 부재로 사생 간 갈등이 빈번했습니다.
3. **정보 공유의 단절:** 사생 공지가 파편화되어 있어 학생들이 중요한 정보를 놓치는 경우가 많았습니다.

=> 모바일 자동화 시스템을 통해 기숙사 생활 편의성을 획기적으로 개선하고자 개발했습니다.
